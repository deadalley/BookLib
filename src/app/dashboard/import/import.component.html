Configure your attributes

To correctly import your books from a spreadsheet file, use the following formatting for the provided attributes:

{{displayAttributesInfo ? 'Hide' : 'Show'}}

  • Title : The title of the book (required).

  • Author: The author of the book (required).

  • Original title: The title in the original title of the book.

  • Publisher The publisher of this version of the book.

  • Pages: Page count of this version of the book. Should be a whole number larger than 0.

  • Year: The year of publication of the book. Should be a whole number larger than 0.

  • Language: The language in which this version of the book was written. The language should be available in the following list:

    • {{language}}

  • Rating: Your personal rating. Should be a whole number between 0 and 5.

  • Owned: Indicates if you own this book or not. Mark your owned books with an 'X', otherwise leave this field blank.

  • Wishlist: Indicates if this book is in your wishlist. Mark the books in your wishlist with an 'X', otherwise leave this field blank.

  • Read: Indicates if you've read this book or not. Mark the books you've read with an 'X', otherwise leave this field blank.

  • Favorite: Indicates if this books if one of your favorites or not. Mark your favorite books with an 'X', otherwise leave this field blank.

  • Image link: Provide an image link to the book cover.
